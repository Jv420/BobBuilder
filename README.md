# BobBuilder

Mineflayer builder bot voor DynathiSMP.

## Installatie

```bash
npm install
```

Maak een .env bestand gebaseerd op .env.example.

## Starten

```bash
npm start
```

## Commands

- !setbuilder
- !builder
- !follow
- !stop
- !status
- !house
- !farm
- !storage
- !shop
- !spawn
- !castle

## Vereisten

- Paper 1.21.11
- FastAsyncWorldEdit
- LuckPerms

Plaats schematics in de server en gebruik namen:
starterhouse, farm, storage, shop, spawn, castle.
